# WorkLedger (Progress Tracker)

**WorkLedger** is a Kotlin Android productivity app for developers and students to track daily work sessions, manage tasks, and stay consistent with personal goals.

---

## 📱 About the Project

WorkLedger helps users log their work progress in a structured way.  
The app is built natively for Android using modern Jetpack libraries and follows a clean modular project layout.

From the repository configuration, this app uses:

- **Kotlin**
- **Jetpack Compose (UI)**
- **Room (local database)**
- **Navigation Compose**
- **Hilt (Dependency Injection)**
- **DataStore (Preferences)**
- **Material 3**
- **Lottie (animations)**

---

## ✨ Core Features

- Track daily work/progress sessions
- Manage and organize tasks
- Persist data locally with Room
- Navigation between screens with Compose Navigation
- Store app preferences with DataStore
- Smooth UI with Material 3 + Compose
- Dependency injection with Hilt

## 🧱 Tech Stack

- **Language:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Architecture Support:** AndroidX Lifecycle
- **Database:** Room (`room-runtime`, `room-ktx`, `room-compiler` with KSP)
- **DI:** Hilt
- **Navigation:** `androidx.navigation:navigation-compose`
- **Preferences:** `androidx.datastore:datastore-preferences`
- **Animations:** `lottie-compose`
- **Build System:** Gradle Kotlin DSL (`.kts`)
